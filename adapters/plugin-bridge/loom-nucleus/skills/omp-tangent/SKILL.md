# OMP Tangent

Use when a useful but tangential task appears and should be delegated or queued instead of blocking the current work.

## Process

1. Name the tangent and why it is out of the current scope.
2. Write a self-contained delegation prompt or a deferred follow-up note.
3. Return to the main task without waiting on the tangent.

## Rules

- Do not claim to launch OMP background agents from this skill; it is delegation-prompt guidance only.
- Do not claim to control OMP runtime state.
- Keep the tangent prompt complete enough to run without the current context.
